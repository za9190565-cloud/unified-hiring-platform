# نظام مرسل الذكي - Mursel Smart Engine

## نظام متابعة الرسائل في الوقت الفعلي

### المميزات الجديدة:
✅ متابعة حالة الإرسال لحظياً
✅ عرض الرسائل المرسلة مع التوقيت والحساب المستخدم
✅ سجل مباشر لجميع العمليات
✅ تحديث تلقائي كل 5 ثواني

### التشغيل:

```bash
cd email-engine
npm install
npm start
```

### الوصول للوحة التحكم:
افتح المتصفح على: http://localhost:3000

### الأقسام:
1. **الإحصائيات** - عرض إجمالي الرسائل وحالة الحسابات
2. **قائمة الإرسال** - متابعة كل رسالة مع الحساب المرسل والوقت
3. **السجلات** - سجل مباشر لجميع العمليات
4. **الإعدادات** - تعديل إعدادات المحرك

### ملاحظات:
- يتم تحديث البيانات تلقائياً كل 5 ثواني
- السجلات تحفظ آخر 100 عملية
- يمكن تصدير البيانات بصيغة CSV
